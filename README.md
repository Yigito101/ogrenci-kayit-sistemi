# ogrenci-kayit-sistemi
Bu proje, eğitim kurumlarında öğrenci verilerini dijital ortamda düzenli bir şekilde tutmak amacıyla C# Form kullanılarak geliştirilmiş bir yönetim sistemidir. Uygulama; yeni öğrenci kaydı oluşturma, mevcut bilgileri güncelleme, listeleme ve kayıt silme gibi temel işlemleri basit bir arayüz üzerinden gerçekleştirmeyi sağlar.
